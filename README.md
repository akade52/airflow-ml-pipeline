# airflow-ml-pipeline
# ML Pipeline в Apache Airflow

## Описание
Пайплайн для обработки данных об автомобилях и обучения модели машинного обучения.

## Результаты обучения модели

| Метрика | Значение |
|---------|----------|
| RMSE | 5865.79 |
| MAE | 2960.83 |
| R² | 0.6159 |

## Задачи DAG
1. **download_data** — загрузка данных из интернета
2. **clear_data** — очистка и предобработка
3. **train_model** — обучение модели SGDRegressor
4. **validate_model** — сохранение артефактов

## Артефакты
- `/tmp/cars.csv` — очищенные данные
- `/tmp/lr_cars.pkl` — обученная модель
